This folder contains the core environment perception implementation stack. It includes the input data (depth, RGB, and segmentation) generated by the prior object detection and semantic segmentation frameworks.

- **m6bk.py** provides a DatasetHandler class for loading and visualizing per-frame autonomous-driving sensor data (RGB, depth, segmentation, object detections, lane points), 
  along with supporting functions for plane fitting and lane-line geometry.
- **My_Environment_Perception_Pipeline.ipynb** is the notebook that implements thg environment perception pipeline, using semantic segmentation and depth data with RANSAC to estimate the drivable ground plane,
  classical CV techniques to detect, merge, and extrapolate lane boundary lines, and 2D object detection filtered by segmentation to compute minimum distance to impact for each detected obstacle.
